# Your Store – Frontend

React (JavaScript) + Tailwind CSS e-commerce UI for the Node/Express API.

## Run

```bash
npm install
npm run dev
```

Runs at **http://localhost:3000**. API is proxied to **http://localhost:4000** (start the server from the project root).

## Features

- **Store:** Home, All products, Product detail, Add to cart, Cart, Checkout, Order success, Order history
- **Auth:** Login, Register (cookie-based)
- **Admin:** Dashboard, Manage products, Add product (with image upload), Manage orders (update status)

## Tech

- React 19, Vite 7, React Router, Axios, Tailwind CSS
